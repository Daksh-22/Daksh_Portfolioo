The leverage point is Passive to Active. Moving one user across it is worth 5 to 7 new bottom of funnel signups in revenue. The screener is the only surface that asks a user to encode a thesis before they buy, and today it is almost untouched.

So the real question: why don't Groww users use the screener?

Five breakages, five fixes
Today's screener
Where Thesis fixes it
1Buried entry point. No screener in the top nav, found only by search.
1Surface it in Explore as a first class destination.
2Wrong scope. Titled "Intraday screener", scary for buy and hold users.
2Stock theses for the long term investor.
3Jargon filters. RSI, MACD, near breakout. All trader vocab, zero fundamentals.
3Plain English Zero debt growth, Steady dividends, Bluechip on sale.
4Column overflow. Truncated headers, unusable horizontal scroll on mobile.
4Card layout 3 key metrics surfaced, "More" expands the rest.
5No template, no save. User starts from a blank filter list, can't return to it.
5Templates first "Save thesis" as the primary action.
